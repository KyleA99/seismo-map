# seismo-map

## About

SeismoMap is a web application that fetches live earthquake data from the USGS API, stores it in a PostgreSQL database, and presents it to users through a dynamic and interactive frontend.

SeismoMap allows for the visualization of dynamic seismic data, and allows the user to output the respective data as JSON or XML.

## Getting Started

```bash
# Clone this repository to your local machine
$ git clone https://github.com/KyleA99/seismo-map

# Navigate into the cloned repository
$ cd seismo-map

# Install dependencies
$ npm install

# Run the app
$ npm start
```

This application requires the following modules:
- [Node.js](https://nodejs.org/en) - at least version >=18.0.0 is required.
